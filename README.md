# Process Scheduling
This program simulates the process scheduling part of an operating system.

## How to Build and Run
1. Clone or download the project

Within the root of the project:

2. Run `make`
3. Run `oss`

## Arguments
```
 -h  Show help.
 -l  Specify the log file. Defaults to 'oss.out'.
 ```
